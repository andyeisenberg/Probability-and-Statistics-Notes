Suppose \(S \sim \Bin(n, p)\text{.}\) DefinitionÂ 2.1.6 describes a binomially distributed random variable as counting the number of occurrences of some event, which either happens or not in each of \(n\) independent trials. \begin{gather*} S = H_1 + H_2 + \dotsb + H_n. \end{gather*}
